## LDI  

What is LDI?  
Load indirect. Load a value from a location in memory into a register  

Encoding:  
`1010`: is the opcode for the instruction -- OP_LDI  
`DR`: 3 bits -- destination register  
`PCoffset9`: immediate value embedded in the instruction
